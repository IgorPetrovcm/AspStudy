Для переадресации применяются классы RedirectResult, LocalReddirectResult, RedirectToActionResult, RedirectToRouteResult

Протокол HTTP поддерживает два типа переадресации:

Постоянная переадресация:
При постоянной переадресации сервер будет отправлять браузеру сатусный код 301. при данном типе переадресации предполагается, что запрашиваемый документ окончально перемещен в другое место. И после получение кода 301 браузер может автоматически настраивать запросы на новый ресурс.

Временная переадресация:
При временной переадресации сервер будет отправлять браузеру статусный код 302. при этом будет считать, что запрашиваемый документ временно перемещен на другую страницу.

В обоих случаях может использоваиться объект RedirectResult

Для постоянной переадресации используют
RedirectPermanent

Для переадресации к локальным адресам используют:
LocalRedirect - для временной переадресации
LocalredirectPermanent - для постоянной переадресации
