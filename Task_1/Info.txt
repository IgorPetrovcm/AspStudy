ASP.Net core mvc - представляет в общем виде построение приложений вокруг трех основных компонентов - Model(модели) - отвечает за работу с данными
View(представления) - определяеи визуальную состовляющую
Cintroller(контроллеры) - логика обработки запросов
схема: mvc.png

Razor Pages - представляеи модель, при котором за обработку запроса отвечают специальные сущности - страницы Razor Pages. Каждую отдельную сущность можно ассоциировать с отдельной веб-сьраницей

ASP.Net Core Web API - представляет реализацию паттерна REST, при котором для каждого типа http-запроса  (GET, POST, PUT, DELETE)
